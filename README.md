The dataset comprises 21,864 rows of transactional data from Gamezone, an online electronics store, with 11 columns capturing key details about purchases, users, and operations. Each row represents an individual order, including fields like USER_ID (unique customer identifier), ORDER_ID (transaction ID), timestamps for purchase and shipping (PURCHASE_TS, SHIP_TS), product details (PRODUCT_NAME, PRODUCT_ID, USD_PRICE), and contextual data such as the PURCHASE_PLATFORM (website/app), MARKETING_CHANNEL (e.g., email, social media), ACCOUNT_CREATION_METHOD (e.g. desktop, mobile, tablet), and COUNTRY_CODE (shipping destination).
